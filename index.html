Crucible-Review-Creator by AlejandroRivera

Automatic Code Review Creator for Atlassian's Crucible product.

This is a fork of the plugin found in: https://plugins.atlassian.com/plugin/details/14896 Since this plugin seems to have been dead for a while, I updated the code to work with the latest version of Crucible available until now (Crucible 2.7.6)
